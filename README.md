# blink leds and run motor with the use of a potentiometer

This is a code for the following given exercise:

we have 2 butons(switches), one potentiometer, some leds and one dc motor
we need to have the following states
when switch2 in in one position the leds need to obey to the potentiometer, blinks one after the other with equal speed
when switch1 is turned the direction of blinking leds should change
when switch2 is turned to the other postition the leds should still blinking the switch1 should have no effect and the motor should turn on and obbey to the potentiometer
